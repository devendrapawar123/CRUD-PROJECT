Project Title: CRUD Operations Web Application
Technology: Python, Django, SQLite, HTML, CSS, Git, GitHub

Description:

Developed a Django-based CRUD web application to manage user/student data efficiently.

Implemented Create, Read, Update, and Delete functionalities using Django models, views, and templates.

Used SQLite database for data storage and handled database migrations.

Designed user-friendly forms with CSRF protection and server-side validation.

Followed MVC (Model–View–Template) architecture for clean and scalable code.

Managed source code using Git and hosted the project on GitHub.

Key Learnings:

Hands-on experience with Django framework

Database operations using ORM

Version control and project deployment on GitHub
